Общее:

1. Переменные задаются без отступов и сначала строки.
2. Условные операторы нужны для создания условия позволяющего не дублировать однотипный код.
3. Для задания кода к каждому блоку условий нужны отступы. Они должны быть равны 4-м пробелам или же 1-му табу.
4. Вспомогательные операторы elif, else не являются обязательными.
5. [11:06] Bool сравнения True, False. Можно комбинировать и сдругими типами данных в условии. Могут заводиться отдельно,
через переменные:
    flag = True
6. Циклы позволяют исполнять код пока соблюдаются условия или заданное число раз.
7. Булевы переменные по умолчанию определяются по умолчанию как:
    1) Пустые bool("") False
    2) Заполненные bool("Abc") True



Синтаксис:

[Условные операторы]

if - оператор если.

elif - оператор "уточнения" идущий после  if. Нужен для задания действий выполняющихся в случае если if не отрабатывает.

else - "замыкающий" оператор. Нужен если данные не отрабатывают по всем заданным условиям

== - сравнение левой и правой части переменной, где они должны быть равны.

!= - сравнение левой и правой части переменных, где левая не должна равняться правой.

>= - больше либо равно.

not - оператор отрицания. Может использоваться для смены полученного условия на противоположное.

and - оператор и. При включении, проверяет полное исполнение всех частей условия.

or - оператор или. При включении, проверяет что хотябы одно из условий выполнено. Если условие сработало остальные уже не проверяются.

Тернарный оператор - позволяет минимизировать кол-во строк кода не нарушая логики. #87, 122

:= - оператор позволяет записать значение в переменную и вернуть его для проверки. #122


[Циклы]

for - цикл позволяет  проитерироваться по массиву. #94

continue - позволяет "скипнуть" итерацию.

break - полная остановка цикла.

while - выполняет код пока выполнено условие.#111
