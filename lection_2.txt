Плюсы python:

1. Интерпретируемый язык - позволяет сразу запускать код без компиляции
2. Совместимый с разными OS 
3. Востребован в разработке и автоматизации. Имеет большую базу поддерживаемых библиотек
4. Динамически-типизированный - не нужно писать тип данных перед переменной. Таким образом в одной переменной могут быть
разные типы данных.

Общее:

1. IDE (Integrated Development Environment) - интегрированная среда разработки. Совмещает в себе такой функционал:
    1) терминал OS
    2) git
    3) виртуальные среды
    4) туториалы по синтаксису языков
